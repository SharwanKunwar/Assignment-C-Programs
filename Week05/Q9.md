## 1.	Write a C program to find sum of natural numbers using recursive function.

```c
#include <stdio.h>

// Recursive function to calculate sum of natural numbers
int sumOfNatural(int n) {
    if (n == 0)
        return 0;
    else
        return n + sumOfNatural(n - 1);
}

int main() {
    int number;

    printf("Enter a positive integer: ");
    scanf("%d", &number);

    if (number < 0) {
        printf("Please enter a non-negative number.\n");
    } else {
        int sum = sumOfNatural(number);
        printf("Sum of natural numbers from 1 to %d is %d\n", number, sum);
    }

    return 0;
}

```
